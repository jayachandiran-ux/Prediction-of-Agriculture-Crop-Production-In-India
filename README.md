# 🌾 Prediction of Agriculture Crop Production in India

This repository contains Exploratory Data Analysis (EDA) performed on five agriculture-related datasets as part of the UCT Internship Project. The objective is to understand the datasets, identify data quality issues, visualize trends, and prepare the data for future machine learning tasks.

---

## 📌 Project Objectives

- Perform data cleaning and preprocessing.
- Identify missing and duplicate values.
- Understand dataset structure and data types.
- Visualize agricultural trends using different plots.
- Analyze production, cultivation cost, yield, and crop-related information.

---
